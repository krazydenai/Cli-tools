# KrazydenAI - macOS

## 🚀 One-Touch Launch

Simply double-click **`launch.command`** to start KrazydenAI!

**That's it!** Everything you need is in this folder.

---

## Requirements

- **Node.js** installed on your system
- Install via Homebrew (recommended):
  ```bash
  brew install node
  ```
- Or download from: https://nodejs.org/

---

## First Run

1. Double-click `launch.command`
2. Press `/` to open the command palette
3. Type `/help` to see all commands
4. Start chatting!

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `/` | Open command palette |
| `Cmd+R` | Search history |
| `Cmd+L` | Clear log |
| `Cmd+C` | Exit |
| `Esc` | Close menus |

---

## Troubleshooting

**"Permission denied" when running**
- Just try again - permissions are auto-fixed on first run!

**Can't double-click it?**
- Open Terminal in this folder and run: `bash launch.command`

---

## Features

- 💬 Chat with local LLM models
- 📚 Knowledge Base (RAG) for context
- 🎨 4 Beautiful themes
- 📝 Full search history & bookmarks
- ⚙️ All settings auto-save locally

---

## Quick Commands

```
/help            Show all commands
/model           Switch models
/kb add          Add to knowledge base
/kb search       Search knowledge base
/settings        Open settings

```
