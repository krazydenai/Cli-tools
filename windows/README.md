# KrazydenAI - Windows

## 🚀 One-Touch Launch

Simply double-click **`launch.bat`** to start KrazydenAI!

**That's it!** Everything you need is in this folder.

---

## Requirements

- **Node.js** installed on your system
- Download from: https://nodejs.org/
- Or install via chocolatey: `choco install nodejs`

---

## First Run

1. Double-click `launch.bat`
2. Wait for the terminal to open
3. Press `/` to open the command palette
4. Type `/help` to see all commands
5. Start chatting!

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `/` | Open command palette |
| `Ctrl+R` | Search history |
| `Ctrl+L` | Clear log |
| `Ctrl+C` | Exit |
| `Esc` | Close menus |

---

## Troubleshooting

**"node is not recognized"**
- Check if Node.js is in path
- Install Node.js from https://nodejs.org/
- Then restart and try again

**Window closes immediately (it shouldn't, but if it does)**
- Right-click `launch.bat` → Edit
- Add this line at the end: `pause`
- Save and try again (will now show any errors)

**Port already in use**
- KrazydenAI uses only local files, no ports needed
