# KrazydenAI - Linux

## 🚀 One-Touch Launch

Open Terminal in this folder and run:
```bash
bash launch.sh
```

**That's it!** Permissions are automatically fixed on first run. After that, you can even double-click `launch.sh`!

---

## Requirements

- **Node.js** installed on your system

**Install on Ubuntu/Debian:**
```bash
sudo apt-get update
sudo apt-get install nodejs npm
```

**Install on Fedora/RHEL:**
```bash
sudo dnf install nodejs
```

Or download from: https://nodejs.org/

---

## First Run

1. Open Terminal in this folder
2. Run: `bash launch.sh`
3. Press `/` to open the command palette
4. Type `/help` to see all commands
5. Start chatting!

(Next time you can just double-click `launch.sh` since it auto-fixes permissions on first run!)

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `/` | Open command palette |
| `Ctrl+R` | Search history |
| `Ctrl+L` | Clear log |
| `Ctrl+C` | Exit |
| `Esc` | Close menus |

## Troubleshooting

**"command not found: node"**
- Install Node.js:
  ```bash
  # Ubuntu/Debian
  sudo apt-get install nodejs npm
  
  # Fedora
  sudo dnf install nodejs
  ```

**"Permission denied"**
- Just try again - permissions are auto-fixed on first run!
